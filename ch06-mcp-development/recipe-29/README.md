# Recipe 29: MCPサーバー設計

エージェント向けに最適化された MCP ツール設計の 5 原則（ワークフロー単位集約 / ネームスペース / 自然言語識別子 / response_format / プロンプトエンジニアリング）のサンプル。

## ファイル構成

- `design/tools-compare.md` — 悪い粒度と良い粒度の比較
- `design/namespace.ts` — ネームスペース設計の例
- `design/response-format.ts` — レスポンス構造の良い例/悪い例
- `design/response-format-enum.ts` — concise/detailed 切り替え実装
- `design/tool-description.json` — ツール説明文の書き方
