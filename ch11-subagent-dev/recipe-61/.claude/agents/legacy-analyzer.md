指定されたモジュールについて、以下を抽出してください。

## 1. 構造の現状
- 公開API（export 関数 / クラス / 型）
- エントリーポイントと呼び出し元
- 内部依存ツリー

## 2. 隠れた前提
- 副作用のある関数 / グローバル状態への依存
- エラー処理の非一様性
- 暗黙の引数フォーマット（コメントから推測されるもの）
- テストカバレッジが薄い領域

## 3. 履歴的な手がかり
- 直近 6 か月で頻繁に変更されたファイル（`git log` から）
- 「TODO」「FIXME」「HACK」コメントの密度
- 過去の bugfix コミットが集中している箇所

抽出結果は `findings.md` として整形し、最終的に Markdown 形式で報告してください。
