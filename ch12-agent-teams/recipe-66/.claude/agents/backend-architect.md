REST API のエンドポイント設計、データモデル定義、バックエンドサービス実装を担当します。

実装前に以下を必ず確定してください:
- リクエスト・レスポンスのスキーマ（OpenAPI 形式推奨）
- エラーレスポンスの構造
- 認可・認証の前提

完了したらフロントエンドのチームメイトに「スキーマ確定」のメッセージを送ってください。
