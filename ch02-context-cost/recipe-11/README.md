# レシピ11: トークン消費を削減するプロンプトテクニック

トークン消費を抑えるための `CLAUDE.md` 設計と、プロンプトテクニックの参考サンプル。

## ファイル一覧

- `CLAUDE.md` — 簡潔な `CLAUDE.md` の例（詳細は Skills へ委譲するスタイル）

## 主なテクニック（本書本文参照）

本書 Ch02 レシピ11 で以下を詳述しています。

1. 具体的なプロンプト（範囲・情報源・既存パターン・症状の明示）
2. `@` によるファイル参照
3. タスク境界での `/clear`
4. サブエージェントへの調査委譲
5. 一時的な質問を `/btw` で文脈外に出す

## 使い方

```bash
# CLAUDE.md をプロジェクトの実態に合わせて書き直す（200 行以下目安）
# 詳細な手順や常用ワークフローは Skills に分離する
```
